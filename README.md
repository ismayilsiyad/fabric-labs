# fabric-labs

Hands-on Microsoft Fabric labs, published at https://ismayilsiyad.github.io/fabric-labs/

- [Create and explore an ontology in Microsoft Fabric](https://ismayilsiyad.github.io/fabric-labs/) — build a Fabric IQ ontology manually from OneLake data (Lakeshore Retail scenario), enrich it with time series data, explore it with graph queries, and consume it with a data agent.
- [Build an ontology from a semantic model in Fabric IQ](https://ismayilsiyad.github.io/fabric-labs/build-ontology-semantic-model.html) — build a Power BI semantic model, generate the ontology from it, then verify, enrich, explore, and consume it with a data agent.
- [Discover and connect to data in OneLake](https://ismayilsiyad.github.io/fabric-labs/discover-onelake.html) — explore the OneLake catalog, create cross-workspace shortcuts, query with the SQL analytics endpoint, and build a semantic model.
